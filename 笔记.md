# 笔记

为什么使用Promise
1.指定回调函数的方式更加灵活
2.支持链式调用,可以解决回调地狱问题

Promise: 启动异步任务 => 返回promise对象 => 给promise对象绑定回调函数(甚至可以在异步任务结束后指定/多个)
回调地狱: 回调函数嵌套调用,外部回调函数异步执行的结果是嵌套的回调执行的条件
